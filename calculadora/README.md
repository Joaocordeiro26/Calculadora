# Calculadora Java — Projeto Integrador IV-A

Projeto desenvolvido para a disciplina de Projeto Integrador IV-A da PUC Goiás.

## Descrição

Módulo de calculadora com as quatro operações matemáticas básicas, com testes unitários implementados usando **JUnit 5**.

## Operações disponíveis

- Somar
- Subtrair
- Multiplicar
- Dividir (lança exceção ao dividir por zero)

## Tecnologias

- Java 11
- Maven
- JUnit 5

## Como rodar os testes

```bash
mvn test
```

## Estrutura do projeto

```
calculadora/
├── pom.xml
└── src/
    ├── main/java/com/projeto/
    │   └── Calculadora.java
    └── test/java/com/projeto/
        └── CalculadoraTest.java
```
